# Rickshaw

Easy way to get a SHA1 hash

## Installation

Add this line to your application's Gemfile:

    gem 'rickshaw'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rickshaw

## Usage

Rickshaw.hash('./file')
